
# Kids Encyclopedia
Little world of knowledge for kids in one app. The kids will able to know about various features of the worlds, such as, continents, countries within each continent, detail information about a country, etc. They can also select their favorite countries and play a simple quiz game where the questions will be based on various information of the favorite countries.
